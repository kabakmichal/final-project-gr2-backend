## Deployment

To deploy this project run

```bash
  npm run start
```

To deploy in dev mode run

```bash
    npm run start:dev
```

## Run Locally

Clone the project

```bash
  git clone https://github.com/kabakmichal/final-project-gr2-backend.git
```

Go to the project directory

```bash
  cd final-project-gr2-backend
```

Install dependencies

```bash
  npm install / npm i
```

Start the server

```bash
  npm run start:dev
```
